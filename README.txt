1) Open the index.html file in a browser
2) Choose username and click login
3) "Search for" allows user to search for tags AND/OR items. The default value is 'All'
4) The filter menu has filter options. Clicking the filter icon (the 'funnel' button beside the green plus) will reveal the filter side bar.
Clicking away from the filter side bar will close it (automatically saving and applying changes)
5) Changing what you serach for (i.e tag or item) will clear filter options as they are dependent on what you are searching for
6) Item cards have images, tag cards begin with a green '#'
